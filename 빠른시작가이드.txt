1. 시작하기
-----------
1) index.html 파일을 더블클릭하여 실행합니다.
2) 웹 브라우저(Chrome, Edge, Firefox 등)에서 자동으로 열립니다.
3) 인터넷 연결이 필요합니다 (Three.js 라이브러리 로딩).

2. 기본 조작법
--------------
- 마우스 왼쪽 버튼 드래그: 3D 뷰 회전
- 마우스 휠: 확대/축소
- 시스템 전환: 상단의 [시스템 비계] / [시스템 동바리] 버튼 클릭

3. 주요 기능
------------
[다음 단계 ▶] : 설치 순서를 단계별로 진행
[◀ 이전 단계] : 이전 단계로 돌아가기
[자동 재생 ▶] : 4초 간격으로 자동 진행
[처음으로]    : 초기 상태로 리셋
[안전 수칙 보기] : 안전 수칙 팝업 표시
[보고서 추출]    : 이미지/문서 추출 기능

4. 보고서 추출 기능
------------------
- 현재 단계 캡처: 현재 화면을 PNG 이미지로 저장
- 전체 단계 캡처: 모든 단계를 이미지로 저장
- 설치 매뉴얼 출력: 텍스트 형식의 설치 가이드 다운로드
- 인쇄용 보고서: 프린터 출력용 문서 생성

5. 활용 팁
----------
- F11키를 눌러 전체화면으로 사용하면 더 몰입감 있게 볼 수 있습니다.
- 각 단계별로 캡처하여 PPT 교육자료로 활용하세요.
- 인쇄용 보고서는 현장 게시용으로 적합합니다.
